Notes of matplotlib

13 Dec, 2015 · by Baisheng Lai · Read in about 1 min · (118 Words)
matplotlib

Here are some notes of matplotlib that are useful. It includes the basic usage that I have ever experisenced.

plot curve

import matplotlib.pyplot as plt
import numpy as np
x = np.arange(0,1,0.1)
y = np.sin(x)
plt.figure()
plt.plot(x, y)
plt.show()

set line properties

Use help(plt.plot) to see what properties are supported, in fact, these properties are very similar to MATLAB, like «linewidth», markers and colors.

plt.plot(x, y, linewidth = 3)
plt.plot(x,y,'r+', markersize = 5)

title and text

One can give title for each plot or subplot, for example

plt.plot(x,y)
plt.title('sin function')

Sometimes, one need to insert a text one figure to indicate some concept(expecially to indicate a point). Note, a coordinate must be assigned.

plt.plot(x,y)
plt.text(1, 1, 'a point')

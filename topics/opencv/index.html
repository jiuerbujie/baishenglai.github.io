DRAFT :: 用cmake管理多版本的OpenCV

11 Dec, 2015

简介 本问介绍如何利用cmake方便地管理多个版本的OpenCV，并且在实际项目中灵活地选用需要的版本。 目录变量 为了叙述方便，定义三个变量，$OpenCV_SRCDIR为OpenCV源代码下载位置，$OpenCV_BUILDDIR为OpenCV编译的目标位置，OpenCV_INSTALLDIR为安装OpenCV的位置。 编译OpenCV 首先，到OpenCV官网下载多个版本的OpenCV源代码，如下载opencv-3.0.0.zip与opencv-2.4.10.zip到目录$OpenCV_SRCDIR，分别解压。 接下来，对两个版本的OpenCV使用cmake进行编译。进入$OpenCV_BUILDDIR文件夹，输入命令: cmake $OpenCV_SRCDIR -D CMAKE_BUILD_TYPE=RELEASE CMAKE_INSTALL_PREFIX=$OpenCV_INSTALLDIR 完成后输入 make make install 可在$OpenCV_BUILDDIR中看到如下文件 | |--OpenCVConfig.cmake |--OpenCVConfig-version.cmake |--OpenCVModules.cmake |--... 并在$OpenCV_INSTALLDIR中看到如下文件夹 |--bin\ |--include\ |--lib\ |--share\ 为项目创建CMakeLists.txt
